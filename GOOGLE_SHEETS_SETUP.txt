1. Open: https://sheets.google.com

2. Klik: "Blank" (leeg spreadsheet)

3. Hernoem het sheet:
   - Klik op "Untitled spreadsheet" bovenaan
   - Type: "Pipedrive JobDigger Deals"
   - Enter

4. Klik op cel A1 (linksboven)

5. OPTIE A - Met Tabs (Makkelijkst):
   - Kopieer de regel hierboven (tussen de lijnen)
   - Plak in cel A1
   - Headers worden automatisch verdeeld over kolommen A t/m N

6. OPTIE B - Handmatig Typen:
   - Type "Deal ID" in A1, druk Tab
   - Type "Title" in B1, druk Tab
   - Type "Status" in C1, druk Tab
   - ... enzovoort voor alle 14 headers

7. Maak header row bold (optioneel maar mooi):
   - Selecteer rij 1 (klik op nummer "1")
   - Klik "B" (bold) in de toolbar

8. Freeze de header row (zodat hij blijft staan):
   - View → Freeze → 1 row

NA HET PLAKKEN - FORMAT DE KOLOMMEN:

1. Kolom D (Value) - Currency Format:
   - Klik op kolom D header (de letter "D")
   - Format → Number → Currency → €EUR

2. Kolommen J, K, L, M, N - Date/Time Format:
   - Selecteer kolommen J t/m N (sleep over de headers)
   - Format → Number → Date time

   - Dubbelklik op een kolom scheidingslijn
